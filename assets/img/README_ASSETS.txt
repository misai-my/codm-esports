Drop your Google Drive assets here.

Recommended filenames used by assets/js/config.js:

1. codm-logo.png
   Optional site/tournament logo.

2. kv-bg.jpg
   Main hero/key visual background.

3. kv-character-left.png
   Left transparent character render.

4. kv-character-right.png
   Right transparent character render.

5. pattern.png
   Optional texture/pattern overlay.

If your downloaded files have different names, either:
- rename them to the recommended filenames, or
- update the ASSETS object in assets/js/config.js.

The site has CSS gradient fallbacks, so it will still work even before you add the images.
